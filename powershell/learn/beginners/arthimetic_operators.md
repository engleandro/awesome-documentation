# ARTHIMETIC OPERATORS

## BASIC OPERATIONS

```shell
# math operators: +, -, *, /, %
2 + 2               # 4 # add/sum (+)
2 - 1               # 1 # minus/subtract (-)
2 * 2               # 4
2 / 2               # 1
2 % 2; 3 % 2        # 0, 1  # rest dividision
( 1 + 1 ) * 5       # 10
```
